## 多数据处理
* 类型最大值/最小值: `#include "limits.h"`
    * 整型最大值: `INT_MAX`
    * 整型最小值:`INT_MIN`

* 万能头文件: `#include <bits/stdc++.h>`

## 累运算
* env: `int sum, a;`
* 累加 (`sum = sum + a`) => `sum += a`
* 连乘 (`sum = sum * a`) => `sum *= a`
* 递减 (`sum = sum - a`) => `sum -= a`
* 连除 (`sum = sum / a`) => `sum /= a`


## Tips
* 注意整数与浮点运算可能会丢失精度
