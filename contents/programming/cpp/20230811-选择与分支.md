## `switch`
* 语法: (其中, `<>` 代表您需填写的内容)
  ```cpp
  switch (<表达式>) {
      case <值1>:
          <程序段>
      case <值2>:
          <程序段>
      ...
      case <值n>:
          <程序段>
      default:
          <均不满足执行的程序段>
  }
  ```

* 在 `switch` 中, 执行 `break` 可跳出 `switch`, 以免其他满足的条件的 `case` 分支执行
<br> `break` 后, 将会运行 `switch` 整个语句体的下一行代码

* 不 `break` 情况下, `switch` 各 `case` 分支从首个符合条件的 `case` 顺序执行, 请注意!
