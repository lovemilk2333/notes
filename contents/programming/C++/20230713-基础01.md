# 20230713 初识 CPP

## 基本类型
| 名称 | 描述 | 所占字节 |
| :---: | :---: | :---: |
| bool | 布尔值 | 1 |
| char | 字符(≠字符串) | 1 |
| ---- | 浮点 | ---- |
| float | (单精度)浮点 | 4 |
| double | 双精度浮点 | 8 |
| long double | 扩展双精度浮点 | 16 |
| ---- | 整形 | ---- |
| short | 短整型 | 2 |
| int | 基本整型 | 4 |
| long long | 长整型 | 8 |

## 运算符
| 符号 | 名称 | 作用 |
| :---: | :---: | :---: |
| + | 加 | 相加 |
| - | 减 | 相减 |
| * | 乘 | 相乘 |
| / | 除 | 相除 <br> `int`或其衍生类型(如`short`) 与 `int`或其衍生类型 相除时仅会保留整数部分, `float`(包括其衍生类型)以此类推 |
